
### プログラミングで使う記号一覧
|  記号  |  読み方  |
| :---: | ---- |
| () | ブラケット、かっこ |
| [] | スクエア、角かっこ |
| {} | カーリー、波括弧 |
| . | ドット,ピリオド |
| , | コンマ,カンマ |
| ; | セミコロン |
| : | コロン |
| ''| シングルクォート |
|""|  ダブルクォート |
|``|  バッククォート |
|~|   チルダ |
|-|   ハイフン |
|_|   アンダースコア |
|<|   小なり（山かっこ） |
|>|   大なり（山かっこ） |
| \| |  パイプ |
| \ | バックスラッシュ |
