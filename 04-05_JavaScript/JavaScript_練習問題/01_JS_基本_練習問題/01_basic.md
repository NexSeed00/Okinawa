2. 変数nameに「SeedKun」を代入し、作成した変数nameと文字を繋げて以下のように表示するプログラムを作成してください。

	出力結果
		
	```
	変数nameの中身：SeedKun
	```
		
	<details><summary>回答例</summary><div>
		
	```
	let name = "SeedKun"
	console.log('変数nameの中身：' + name);

	もしくは

	console.log(`変数nameの中身：${name}`);
	```
		
	</div></details>

	<br>

3. 変数lastNameと変数firstNameを作成し、任意の文字列を代入してください。

	<details><summary>回答例</summary><div>
		
	```
	let lastName = "山田";
	```
