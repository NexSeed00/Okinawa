19. 変数xに「男」または「女」を代入し、以下の表通り出力するプログラムを作成してください。   
（※switch文を使用すること）

 | xの値    | 出力内容 |
 | -------- | -------- |
 | 男       | male     |
 | 女       | female   |
 | 上記以外 | 性別不詳 |

  <details><summary>回答例</summary><div>

  ```

	let x = '男'  
	switch (x) {
		case '男':
			console.log('male');
			break;
		case '女':
			console.log('female');
			break;
		default:
			console.log('性別不詳');
			break;
	}

  ```	
  </div></details>
